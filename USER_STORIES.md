# User Stories

## 1) Home Tab (Marketing "Wow" Page)
- As a first-time visitor, I want a visually compelling Home tab so I immediately understand EasyExtract's value and trust the product.
- As an evaluator, I want clear calls to action from Home (go to workspace, build template) so I can start in one click.
- As a stakeholder, I want a polished narrative (problem, workflow, proof) so I can confidently demo the app.

## 2) Global Jobs Workspace
- As an operations user, I want a single Jobs view across all buckets so I can monitor document processing in one place.
- As a reviewer, I want filters (status, search, bucket, template, date range) so I can quickly find specific jobs.
- As a power user, I want batch actions (re-extract failed jobs, export selected jobs) so I can manage high volume efficiently.

## 3) Side-by-Side Review
- As a QA reviewer, I want to compare source content and extracted output side-by-side so I can validate extraction quality faster.
- As an analyst, I want to switch between visual page preview and OCR text so I can verify both layout and raw content.
- As a workflow owner, I want review accessible from both Bucket and Jobs views so verification is always one click away.
