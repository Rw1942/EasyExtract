# Product Summary (Current)

Last updated: 2026-03-18 20:19:29 MDT

This file is now a lightweight product reference for the current app.

## Core User Flows
- Create a template (manual or AI builder)
- Create a bucket and assign a template
- Upload PDFs or document images
- Run extraction and review results
- Export results to CSV

## Main Views
- Home: marketing/overview page
- Buckets: grouped document processing
- Jobs: global queue, filters, batch actions
- Review: side-by-side source vs extracted values
- Templates: create/edit/duplicate/manage extraction schemas
- Settings: extraction prompt, builder prompt, OCR batch size

## Intended Use (Current Phase)
- Limited/internal usage
- Fast iteration over perfect architecture
- Simple deployment and operations via Wrangler + Cloudflare
